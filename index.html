DIGM Attendance App

 September 2014 - June 2015

Drexel University is a well-known university in the Philadelphia area, providing experiential learning to students from all walks of life. Its Digital Media program is expanding greatly; in the coming year of 2015-2016, the introductory class UNIV 101 will total around 120 students. The professors who run these courses need a more efficient method of taking attendance.

This system will require students to scan their IDs with a card reader, which will record their data (name, ID number) in a database. Faculty will then be able to access and edit the information through an administrative panel they can access by logging into the site. In addition, the project will provide feedback to students, so they know whether they've successfully signed in.

Dota 2 - The International 4 Digital Poster (Warning: Sound)

 April 2014 - June 2014

The International is an video game tournament with teams participating from various countries around the world, most notably China and the United States.

The web page was an exercise to experiment with responsive page layouts and scalable vector graphics (SVGs). CSS transitions and animations were explored as well as the HTML5 <audio> element, which was controlled with JavaScript.

Visual Custom Kick Interface Redesign

 2008 - 2009

Visual Custom Kick (VCK) was a third-party Warcraft 3 tool to improve players' custom game experiences. Warcraft 3 is a medieval-fantasy real-time strategy video game made by Blizzard Entertainment.

Flat UI Color Picker

 2014-11-18

FlatUIcolorpicker is a small project with FlatUI colors. Its intent was to inspire designers to use the colors of Flat UI. It offered a responsive page layout but did not offer much in functionality.

This reimplementation's goal was to not only make a replica of it but to add additional functionality such as easier color code selection when clicking a color box.
